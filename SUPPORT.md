# Support

Use [GitHub Issues](https://github.com/krajcik/phasemill/issues) for reproducible
bugs, feature requests, installation problems, and documentation questions.

Include the Phasemill version, Codex surface and version, operating system,
reproduction steps, expected behavior, actual behavior, and relevant redacted
logs. Never include access tokens, credentials, private source code, personal
data, or unredacted environment output.

Security vulnerabilities must not be filed as public issues. Follow
[SECURITY.md](SECURITY.md) instead.

Support is community-based and has no guaranteed response or resolution time.
