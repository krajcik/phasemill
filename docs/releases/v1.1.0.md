# Phasemill v1.1.0

Phasemill is now published as an independent Codex plugin with a durable,
restart-safe workflow for design, planning, implementation, native review,
optional Pi/GLM review, finalization, and proposal-only project learning.

## Highlights

- learn durable guidance from current-session corrections and verified accepted
  review comments in one explicitly named pull request
- propose project `.codex/phasemill/` or explicitly requested user-global
  `PLUGIN_DATA` changes without writing them automatically
- require candidate selection and approval of a fresh exact diff before any
  learning update
- retain scoped Go, Python, PHP, Java/Kotlin, JavaScript/TypeScript, and Rust
  review profiles
- publish pinned marketplace installation, branding, support, security,
  privacy, release, and submission documentation

See [CHANGELOG.md](https://github.com/krajcik/phasemill/blob/v1.1.0/CHANGELOG.md)
for the full compatibility and safety notes.
